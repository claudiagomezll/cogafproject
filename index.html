iPlus Extension

Este proyecto presenta una solución que transforma modelos UML en código Java funcional mediante el uso de Acceleo.


Acceleo, una poderosa herramienta de generación de código basada en plantillas, se convierte en el puente que vincula la representación gráfica de un sistema en UML con la implementación detallada en Java. Este proceso de transformación automatizada simplifica la transición del diseño conceptual a la realidad del código, ahorrando tiempo y reduciendo errores potenciales.

Systematic Literature Review

La página web asociada proporciona una interfaz intuitiva para cargar modelos UML y, a través de interacciones simples, invoca el proceso de generación de código con Acceleo. El resultado es un archivo comprimido que contiene el código Java generado, listo para su utilización y personalización en el desarrollo de aplicaciones.
